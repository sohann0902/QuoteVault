A simple, clean iOS Quotes app built with SwiftUI, Core Data, and WidgetKit.

## Features
- Random quote fetch (API Ninjas)
- Save, favorite, share quotes
- Core Data persistence
- Widget shows favorite quote (refreshes daily)

## Tech
- SwiftUI + MVVM
- Async/Await networking
- Core Data
- WidgetKit + App Group

## Setup
1. Add your API key in `BrewApps/APIConfig.swift`.
2. Enable App Groups for app + widget using `group.com.Sohan.BrewApps`.
3. Run the app, save a quote, and favorite it for the widget.

## Widget
- Small + Medium supported
- Shows a favorite quote, or empty-state text if none
